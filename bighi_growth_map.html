Win-back · M14
Day 45 Silence Trigger
If a customer hasn't reordered 45 days after purchase (and soap should be finished), fire a single win-back message: a simple question, not a discount. Ask what happened. The reply rate is 30–40% and the insight is invaluable. Discounts come only at Day 60 and only if Day 45 got no response.
"Don't panic-discount at Day 45. Ask first. Understand churn before paying to reverse it."
1-day build · Make.com condition branch

System 06 · Infrastructure
Data Capture Operating System
All Touchpoints

Every piece of captured data should map to a downstream action. If it doesn't, don't capture it. Here's the complete map of what to capture, where, and what it unlocks.

📱 Phone Number (WhatsApp) Checkout (Step 1) · Scroll CTA bar · Skin quiz Entire post-purchase sequence · Refill nudges · Broadcasts 🔴 P0 — Without this, nothing works
🧴 Skin Type Hero pill selector · Skin quiz Personalized product recommendations · Seasonal variant targeting 🟠 P1 — Enables personalization
🎯 Primary Concern Skin quiz (Q2) Routine map output · Product page messaging · WA content personalization 🟠 P1 — Drives routine relevance
🛒 Products Owned Purchase history (Razorpay webhook) "Completes Your Routine" module · Refill countdown per product 🔴 P0 — Core of reorder system
📅 Order Date Razorpay webhook → Airtable Triggers Day 7/14/22/30 WhatsApp sequence · Refill countdown 🔴 P0 — All automation depends on this
💬 WA Reply Behavior Interakt inbound (replies to messages) Churn prediction · Engagement score · Skip/Pause handling 🟢 P2 — Improves system over time
👁 Product Page Views JS event on 3s+ page dwell Interest signal → WhatsApp follow-up if no purchase in 48h 🟢 P2 — Advanced retargeting
📊 Routine Completion % localStorage routine state → server on checkout "Complete Your Ritual" upsell personalization · Subscription candidate signal 🟢 P2 — Segmentation
The Minimum Viable Data Stack: Phone number + Order date + Products ordered. These three fields, stored in an Airtable table keyed by phone number, unlock the entire refill automation system. Everything else is an upgrade.
